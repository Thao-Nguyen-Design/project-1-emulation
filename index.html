Drawing Machine

I have always found reflection and
symmetry to be magical and mysterious.
We as humans can never make something
perfectly symmetrical without the help
of tools, hence I want to create a drawing
machine that enable us to do that.

Using this tool, you can create your own
unique symmetrical design. It has element
of randomness, but also provide the users
with some options for customization.
